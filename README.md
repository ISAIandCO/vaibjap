# vaibjap — Nihongo Quest

Сайт на GitHub Pages для изучения японского с упором на формат ежедневных коротких сессий.

## Что реализовано

- PWA: `manifest.webmanifest`, `sw.js`, офлайн-кэш app shell.
- 42 урока (6 недель, больше месяца), каждый рассчитан примерно на 6 минут.
- Формат упражнений в стиле языковых тренажёров:
  - выбор ответа;
  - чтение каны;
  - перевод;
  - сборка фразы из токенов;
  - аудио-задания через SpeechSynthesis;
  - «boss-round» на закрепление.
- Система прогресса: XP, серия дней (streak), завершённые уроки, ежедневные квесты.
- Все данные прогресса хранятся в `localStorage`.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`.
