# vaibjap — Nihongo Quest

PWA-сайт для изучения японского на GitHub Pages.

## Что реализовано

- Полноценный PWA-режим:
  - `manifest.webmanifest`
  - `sw.js` с кэшированием app shell и офлайн-поддержкой.
- Курс на 42 дня (6 недель, больше месяца).
- Каждый урок рассчитан минимум на 5 минут (обычно 8 минут).
- Уроки открываются постепенно: следующий доступен только после завершения предыдущего.
- Есть отдельные уроки повторения (каждый 7-й день).
- Педагогическая последовательность в уроке:
  1) сначала обучение буквам,
  2) затем обучение словам и их переводу,
  3) затем задача «услышь слово → выбери правильное написание»,
  4) только после этого вопрос на перевод,
  5) грамматика сначала объясняется, а потом задаётся вопрос по ней.
- Для задания с выбором правильного предложения слова и часть букв заранее вводятся в уроке.
- В КАЖДОМ задании есть возможность прослушать произношение (`🔊 Прослушать`).
- Повторное нажатие на `🔊 Прослушать` запускает замедленное воспроизведение.
- Задания на сбор слова работают и на смартфоне: добавлен tap-to-place (нажатие по букве), помимо drag-and-drop.
- Ошибки не исправляются во время выполнения:
  - ответ сохраняется сразу;
  - проверка и список ошибок показываются после урока;
  - доступна "пересдача" только ошибочных заданий.
- Сохраняется прогресс (`localStorage`): XP, серия дней, завершённые уроки.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`.
