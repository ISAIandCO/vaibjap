# vaibjap — Nihongo Quest

PWA-сайт для изучения японского на GitHub Pages.

## Что реализовано

- Полноценный PWA-режим (`manifest.webmanifest`, `sw.js`, install-иконки).
- Курс на 42 дня с последовательным открытием уроков.
- Педагогическая последовательность:
  1) обучение буквам,
  2) обучение слову + переводу,
  3) аудио-задание «услышь и выбери написание»,
  4) только после этого вопрос на перевод.
- Грамматика расширена по курсу:
  - есть отдельные обучающие grammar-шаги,
  - в объяснении есть и правило, и письменный пример предложения, и пояснение.
- В заданиях на сбор слова при выборе/перетаскивании буквы воспроизводится её произношение.
- Добавлены и сохранены задания «аудио -> выбрать написание на японском».
- Повторное нажатие на `🔊 Прослушать` включает замедленное воспроизведение.
- Частотная лексика:
  - основной источник слов: `https://raw.githubusercontent.com/ISAIandCO/vaibjap/refs/heads/main/wikiwords.lst`;
  - fallback: локальный `wikiwords.lst`, затем встроенный словарь при ошибке.
- Прогресс хранится в `localStorage` (`nq_progress`, `nq_stats`).

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`.
