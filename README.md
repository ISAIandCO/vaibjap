# vaibjap — Nihongo Quest

PWA-сайт для изучения японского на GitHub Pages.

## Что реализовано

- Полноценный PWA-режим:
  - `manifest.webmanifest`
  - `sw.js` с кэшированием app shell и офлайн-поддержкой.
- Курс на 42 дня (6 недель, больше месяца).
- Каждый урок рассчитан минимум на 5 минут (в среднем 6–7 минут).
- Уроки открываются постепенно: следующий доступен только после завершения предыдущего.
- Есть отдельные уроки повторения (каждый 7-й день).
- В каждом уроке:
  - задания на чтение и перевод;
  - аудио-кнопка для прослушивания произношения букв и слов;
  - задание на составление слова перетаскиванием букв на поле.
- Ошибки не исправляются во время выполнения:
  - ответ сохраняется сразу;
  - проверка и список ошибок показываются после урока;
  - доступна "пересдача" только ошибочных заданий.
- Сохраняется прогресс (`localStorage`): XP, серия дней, завершённые уроки.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`.
