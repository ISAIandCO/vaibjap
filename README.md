# vaibjap — Nihongo Quest

PWA-сайт для изучения японского на GitHub Pages.

## Что реализовано

- Полноценный PWA-режим:
  - `manifest.webmanifest`
  - `sw.js` с кэшированием app shell и офлайн-поддержкой.
- Курс на 42 дня (6 недель, больше месяца), уроки открываются последовательно.
- В уроке соблюдена последовательность: буквы → обучение слову/переводу → аудио-распознавание написания → проверка перевода.
- Грамматика сначала объясняется (обучающий шаг), потом задаётся вопрос по ней.
- Повторное нажатие на `🔊 Прослушать` включает замедленное воспроизведение.
- Сбор слова работает и на смартфоне (tap-to-place + drag-and-drop).
- Частотная лексика:
  - приложение пытается загрузить top-10000 японских слов из открытого частотного списка (`ja_50k`) и использует их в генерации уроков;
  - при недоступности сети/источника используется локальный резервный словарь.
- После урока показываются ошибки и доступна «пересдача» только ошибочных заданий.
- Прогресс сохраняется в `localStorage` (`nq_progress`, `nq_stats`).

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`.
